|mongodb| inMemory Dashboard
================================================================================

The |mongodb| inMemory dashboard shows statistics on the |inmemory|
storage engine for the selected |mongodb| instances. This dashboard
contains the following metrics:

.. hlist::
   :columns: 2

   - InMemory data size
   - InMemory max data size
   - InMemory available
   - InMemory dirty pages
   - InMemory transactions
   - InMemory capacity
   - InMemory sessions
   - InMemory pages
   - InMemory concurrency tickets
   - Queued operations
   - Document changes
   - InMemory cache eviction
   - Scanned and moved objects
   - Page faults
