# Time zone data not loaded

## Description

Checks if the MySQL time zone data is loaded or not.
For more information, see [MySQL Server Time Zone Support](https://dev.mysql.com/doc/refman/8.4/en/time-zone-support.html) in the MySQL documentation.

## Resolution

Consider loading time zone data.
